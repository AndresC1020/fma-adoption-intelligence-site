Evaluación

Tabla de requerimientos completa

La rúbrica del proyecto diligenciada en su totalidad: los 21 requerimientos definidos en la Entrega 1, cada uno con la prueba realizada, el criterio de cumplimiento, la evidencia observada, los ajustes implementados durante la validación y la justificación correspondiente. Estado general: 17 requerimientos cumplen plenamente  ·  3 cumplen de forma parcial con justificación documentada  ·  1 no cumple (R11: sobreajuste por muestra pequeña, con plan correctivo para Q3-Q4 2026).

Prototipo

Entregables del prototipo

El sistema está compuesto por un pipeline de datos, un modelo predictivo, una interfaz interactiva y documentación técnica y de usuario. Todos los componentes son accesibles desde este sitio o desde el repositorio público de GitHub.

Ejecutable

Prototipo interactivo

Tablero con las tres vistas analíticas: Factores de adopción (descriptivo), Puntuación de riesgo (predictivo) y Guía de migración (prescriptivo). Desplegado en Streamlit Community Cloud. No requiere instalación ni cuenta.

Ver demo e instrucciones de acceso
Código fuente

Repositorio GitHub

Código completo del proyecto: pipeline.py (ETL + modelo + SHAP), app.py (tablero Streamlit), módulos src/etl.py y src/config.py, modelo serializado y datos de muestra. Incluye instrucciones de instalación local.

Ver código fuente y estructura
Manual

Manual de usuario

Guía completa de uso del tablero: cómo acceder, estructura de las tres vistas, uso de filtros interactivos, interpretación del score de riesgo y los factores SHAP, proceso de actualización de datos y glosario de términos.

Ver manual de usuario
Arquitectura

Pipeline y arquitectura

Diagrama de flujo completo del sistema en dos filas: desde las fuentes de datos (Snowflake / CSV) hasta el tablero en el navegador. Descripción del stack técnico (pandas, scikit-learn, SHAP, Streamlit) y las cinco fases del pipeline.

Ver arquitectura del pipeline
Modelo

Modelo predictivo

Random Forest con validación out-of-time. Métricas: AUC-ROC = 0.829, F1 = 0.82, Sensibilidad = 0.78, Precisión = 0.86. Comparación con tres modelos candidatos y justificación de la selección. Factores SHAP más determinantes.

Ver modelo predictivo
