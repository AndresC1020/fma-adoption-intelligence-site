Repositorio GitHub

Código completo del pipeline de datos, modelo predictivo y tablero Streamlit.

Acceso al repositorio

Código completo y abierto

El repositorio contiene todo el código necesario para replicar el análisis, re-entrenar el modelo y desplegar el tablero en local o en la nube.

Ver repositorio en GitHub ↗

app.py

Aplicación principal de Streamlit. Las tres vistas analíticas (Factores, Riesgo, Guía) y la lógica de navegación entre ellas.

pipeline.py

ETL, ingeniería de características, scoring RF y cálculo de SHAP values. Cinco fases secuenciales. ~21 seg sobre muestra.

src/

Módulos de soporte: etl.py (transformaciones, feature store) y config.py (rutas y parámetros globales).

models/

rf_model.joblib — pipeline scikit-learn serializado. rf_model_metadata.json — features y métricas del modelo.

data/

Archivos parquet de salida del pipeline: analytical_dataset, shap_values, feature_importance, tenant_scores.

documentacion/

Manual de usuario y documentación técnica en markdown. Base de los documentos R3 y R4 de la rúbrica.

Ejecución local

Cómo correr el proyecto

Requiere Python 3.9+. Los datos de muestra están incluidos en el repositorio (n=140 técnicos, 54 clientes).

# 1. Clonar el repositorio
git clone https://github.com/mbayonal/fma-adoption-intelligence.git

# 2. Crear y activar entorno virtual
python -m venv .venv
source .venv/bin/activate # macOS/Linux
.venv\Scripts\Activate.ps1 # Windows PowerShell

# 3. Instalar dependencias
pip install -r requirements.txt

# 4. Generar datos analíticos (~21 seg con datos de muestra)
python pipeline.py

# 5. Lanzar el tablero
streamlit run app.py
# Abre en http://localhost:8501
